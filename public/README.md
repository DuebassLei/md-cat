# 静态资源目录

此目录用于存放项目的静态资源文件。

## 目录结构

- `images/` - 存放图片资源（头像、图标等）
- `qrcodes/` - 存放二维码图片

## 使用方式

### 在代码中引用

在 Vue 组件中，可以通过以下方式引用 public 目录下的资源：

```vue
<!-- 图片 -->
<img src="/images/avatar.png" alt="头像" />

<!-- 二维码 -->
<img src="/qrcodes/wechat.png" alt="微信二维码" />
```

### 注意事项

- `public` 目录下的文件会被直接复制到构建输出的根目录
- 引用时使用绝对路径 `/` 开头
- 文件名建议使用小写字母、数字和连字符
